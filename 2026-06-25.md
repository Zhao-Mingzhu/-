## 七、经验总结

### Vibe Coding 闭环流程
```
描述需求 → AI生成 → 运行验证 → 问题归因 → 迭代提示词 → AI修正 → 验证通过
```

### 常见问题清单
1. **依赖版本问题**：uni-app npm包版本不存在 → 改用Vue3+Vite
2. **MQTT库版本**：paho-mqtt 2.0参数变化 → 降级到1.x
3. **数据不匹配**：设备MAC配置错误 → 核对数据库实际数据
4. **时区问题**：UTC时间未转换 → API层统一转换

### 高效迭代技巧
1. 先验证核心功能（数据流转），再优化细节（界面、时区）
2. 使用启动脚本一键启动所有服务，减少手动操作
3. 数据库查询前先检查实际数据内容
4. 日志是定位问题的关键工具
